# LearningNotes
這裡會存放我的一些學習筆記，主要是 C# 與 .NET 技術。

這裡的筆記通常是我在公開發表於部落格之前的初稿，但也可能是持續修訂潤飾的結果。如有任何建議或補充，歡迎透過 GitHub 的 [Issues](https://github.com/huanlin/LearningNotes/issues) 功能提問或建立工單，也歡迎發 Pull Request 給我。

Happy learning!
